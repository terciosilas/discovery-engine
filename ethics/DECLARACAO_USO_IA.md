# Declaração de Uso de Inteligência Artificial

> Projeto: Discovery Engine
> Data: 2026-03-10
> Pesquisador: Tércio Silas Azevedo

---

## Ferramentas de IA utilizadas neste projeto

| Ferramenta | Uso | Versão |
|---|---|---|
| **Claude Code (Anthropic)** | Orquestração do projeto, geração de código, análise exploratória | Claude Opus 4.6 |
| **Claude API (Anthropic)** | Resumo automatizado de papers, extração de informações | A definir |
| **Google Colab** | Ambiente de execução para processamento pesado | Runtime padrão |

## Como a IA foi utilizada

1. **Geração de código:** Scripts de coleta, processamento e análise foram gerados com assistência do Claude Code e revisados pelo pesquisador.
2. **Resumo de papers:** Abstracts e textos completos de papers open access foram processados via API do Claude para geração de resumos estruturados.
3. **Análise exploratória:** IA foi utilizada para identificar padrões iniciais nos dados e sugerir direções de pesquisa.
4. **Escrita:** Rascunhos de seções do paper foram gerados com assistência de IA e extensivamente editados pelo pesquisador.

## O que a IA NÃO fez

- A IA **não tomou decisões** sobre inclusão/exclusão de papers ou dados
- A IA **não interpretou** resultados — todas as interpretações são do pesquisador
- A IA **não fabricou** dados ou resultados
- A IA **não substituiu** a revisão crítica humana

## Transparência

Todo código utilizado está disponível no repositório do projeto (GitHub privado, disponível mediante solicitação). Os logs de execução em `audit_logs/` documentam cada interação com APIs de IA.

Esta declaração segue as diretrizes da COPE (Committee on Publication Ethics) e as políticas de uso de IA dos principais journals da área.
